# 🎮 STICK FARM PRO ULTRA

Повнофункціональна онлайн гра з реал-тайм чатами, рулеткою та Tic-Tac-Toe!

## ✨ НОВІ ФУНКЦІЇ v2.0

### 📱 Реал-тайм Система Чатів
- ✓ Миттєве оновлення повідомлень без перезавантаження
- ✓ Лічильник нових повідомлень
- ✓ Плавні анімації при прибутті повідомлень
- ✓ Автоскролл до останнього повідомлення
- ✓ Можливість писати нові повідомлення під час гри

### 👥 Улучена Система Друзів
- ✓ Реал-тайм запити на дружбу з модальними вікнами
- ✓ Лічильники запитів і друзів на вкладці
- ✓ Статус друзів (онлайн/офлайн)
- ✓ Один клік для открыття чату

### 🎰 Покращена Рулетка
- ✓ **Розміри сегментів відповідають шансам:**
  - 🔴 RED: 48% = 172.8° (найбільший)
  - ⚫ BLACK: 24% = 86.4° (найменший)
  - 🟢 GREEN (0): 28% = 100.8° (середній)
- ✓ Швидша анімація (0.8s)
- ✓ Відображення відсотків на кнопках
- ✓ Гарніші відтінки кольорів

### 🎮 Онлайн Tic-Tac-Toe
- ✓ Відправлення викликів друзям з ставками
- ✓ Модальне вікно з прийняттям/відхиленням
- ✓ Реал-тайм оновлення ігрового поля
- ✓ Поддержка ставок STICK монет

### 🖼️ Покращений Дизайн
- ✓ **Вибір аватара з галереї** - завантажуйте власні фото!
- ✓ Сучасна градієнтна схема
- ✓ Сповіщення з анімаціями
- ✓ Адаптивний дизайн для мобільних

### ⚡ Оптимізація
- ✓ WebSocket з 100ms затримкою
- ✓ Автосавування кожні 5 секунд
- ✓ Реконекшн при розриві зв'язку
- ✓ Кешування даних на клієнті

## 🚀 ВСТАНОВЛЕННЯ

### 1. Клонуємо репозиторій
```bash
git clone https://github.com/oleksandrenkosasa2-coder/Stick_coin-demo-2.git
cd Stick_coin-demo-2
```

### 2. Встановлюємо залежності
```bash
npm install
```

### 3. Запускаємо сервер
```bash
npm start
```
Сервер запуститься на `http://localhost:3000`

### 4. Відкриваємо клієнт
Відкрийте `index-online-improved.html` у браузері

## 🎮 ГАЙД КОРИСТУВАЧА

### Реєстрація & Вхід
1. Виберіть "РЕЄСТР" для нової гри
2. Введіть логін та пароль (мін. 4 символи)
3. Прийдіть до друзів в системі!

### Чати
- Кліком на друга відкриється приватний чат
- Вводьте повідомлення в реал-тайм
- Нові повідомлення приходять МИТТЄВО 💬

### Друзі
- Пошук користувачів у полі "Логін..."
- Прийміть запити - вони приходять автоматично 📨
- Кліком на друга - привіт в чаті! 👋

### Рулетка 🎰
- Поставте на червоне (48%), чорне (24%) або зелене (28%)
- Множник: RED/BLACK = 2x, GREEN = 14x
- Розміри сегментів **пропорційні шансам**!

### Tic-Tac-Toe Онлайн 🎮
1. Кліком "ВИКЛИКАТИ НА ГРУ"
2. Оберіть противника та ставку
3. Противник отримає сповіщення
4. Грайте онлайн в реал-тайм!

### Аватар 🖼️
- Перейдіть до "⚙️ НАЛАШТУВАННЯ"
- Кліком на аватар - виберіть фото з галереї
- Натиніть "ЗБЕРЕГТИ"
- Друзі побачать ваше фото!
